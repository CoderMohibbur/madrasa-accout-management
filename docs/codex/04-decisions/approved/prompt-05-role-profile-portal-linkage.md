# Prompt 05 Role Profile Portal Linkage

Approved baseline decisions from prompt-05:

- Role membership expresses potential domain access only; it is not the final portal gate.
- Donor portal access is donor-domain-only and independent from guardian linkage.
- Guardian informational access and guardian protected access are separate authorization boundaries.
- Guardian linkage is the additional requirement for protected student-owned data access.
- Shared multi-role home behavior must derive from eligible contexts, not raw role ordering.
