# PHASE 3 — DONOR PORTAL

## Objective

Build a safe donor portal for self-service viewing and controlled donation initiation boundaries, still without full live gateway finalization unless Phase 5 prerequisites are already satisfied.

## Scope

- donor route group
- donor dashboard
- donation history
- receipt history
- optional fund/campaign browsing if additive and approved

## Hard Restrictions

- donor must access only own records
- no unsafe reuse of legacy manual donation logic
- no broad management UI rewrites
